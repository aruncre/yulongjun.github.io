运算符分为以下几种类型：

1 算术运算符

算术运算符 说明 示例
+ 加法 5+8=13
- 减法 90-10=80
* 乘法 4*7=28
/ 浮点数除法 7/2=3.5
// 整除，或叫地板除 7//2=3
% 模，即求余数 7%3=1
** 幂 3**4=81

2 比较运算符

比较运算符 说明 示例
> 大于 3 > 2 为True
< 小于 3 < 2 为False
== 等于 3 == 3为True
!= 不等于 3 != 3为False
>= 大于等于 5>=4为True
<= 小于等于 6<=3为False

3 逻辑运算符

比较运算符 说明 示例
and 与。两边都是真值，返回后面那个值,否则返回False 3 and 5 返回5，5 and 0返回False，True and False 返回False
or 或 。两边有一个真值，返回那个真值，否则返回False 5 and 0返回5，True and False返回False
not 非。非真为假，非假为真 not 5为True，not 0为False，not True为False，Not False为True

5.1 赋值运算符

=：把右边赋值给左边，例如a = 5

a += b、a -= b、a *= b、a /=b、a //=b、a %= b、a **= b：相当于a = a + b、a = a - b…

5.2 成员运算符

in 和not in：一个元素是否存在于一组元素中。后续会讲到，略

5.3 身份运算符

is和not is：两者是否是指向同一个内存对象。
