3.8 集合（Sets）

发表于 2016-07-16   |   分类于 Learning-Python3   |  

Python中的集合是一组可哈希的值的集合。特点：

  • 元素是唯一的。
  • 元素是无序的。

3.8.1 定义一个集合

定义一个set很简单，如果有重复元素，会自动去重：

3.8.2 增

add

增加一个元素到集合，如果元素已经存在，则没有任何操作。

add(…)
Add an element to a set.
This has no effect if the element is already present.

update

增加一组元素，重复的元素会去重

update(…)
Update a set with the union of itself and others.

3.8.3 删

remove

从集合里删除一个元素，元素必须存在，删除不存在的会抛出异常。

remove(…)
Remove an element from a set; it must be a member.
If the element is not a member, raise a KeyError.

discard

从集合中删除一个元素，删除不存在的不会抛出异常。

discard(…)
Remove an element from a set if it is a member.
If the element is not a member, do nothing.

pop

删除并返回一个任意元素。

pop(…)
Remove and return an arbitrary set element.
Raises KeyError if the set is empty.

clear

删除所有元素

clear(…)
Remove all elements from this set.

3.8.4 改、查

没有一个方法可以直接的修改集合中的某个具体元素，因为没有一个方法，可以查找定位其中的某个具体元素。

3.8.4 集合的集合运算

并集（union==|）

求两个集合的并集

union(…)
Return the union of sets as a new set.
(i.e. all elements that are in either set.)

交集（intersection==&、intersection_update）

求两个集合的交集。

一种不改变前面的集合，一种带update改变前面的集合。

intersection(…)
Return the intersection of two sets as a new set.
(i.e. all elements that are in both sets.)

intersection_update(…)
Update a set with the intersection of itself and another.

差集（difference==-、difference_update）

求两个集合的差集。

一种不改变前面的集合，一种带update改变前面的集合。

difference(…)
Return the difference of two or more sets as a new set.
(i.e. all elements that are in this set but not the others.)

difference_update(…)
Remove all elements of another set from this set.

对称差集（symmetric_difference==^、symmetric_difference_update）

求两个集合的对称差集（对称差集，把a.difference(b)和b.difference(a)的值union起来）。

一种不改变前面的集合，一种带update改变前面的集合。

symmetric_difference(…)
Return the symmetric difference of two sets as a new set.
(i.e. all elements that are in exactly one of the sets.)

symmetric_difference_update(…)
Update a set with the symmetric difference of itself and another.

是否超集（isuperset）

用来判断前面是否是后面的超集。

issuperset(…)
Report whether this set contains another set.

是否子集(issubset)

用来判断前面是否是后面的子集。

issubset(…)
Report whether another set contains this set.

是否不相交（isdisjoint）

用来判断两个集合是否是不相交

isdisjoint(…)
Return True if two sets have a null intersection.

复制（copy）

复制集合，本质上是再复制出一段内存。

copy(…)
Return a shallow copy of a set.
