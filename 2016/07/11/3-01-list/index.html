3.1 列表

发表于 2016-07-11   |   分类于 Learning-Python3   |  

初始化列表：

下标/索引操作

  • Python中的索引，从前往后，索引是从0开始，从后往前，索引是从-1开始。

  • 如果索引超出范围，将引发IndexError异常。

  • 更改列表中元素的值，采用lst[index] = new_value，索引可用正数，也可用负数，但是不能超出范围，否则也会抛出IndexError异常。

示例：

给list增加元素

append

在列表最后增加一个对象，返回值为None 。

append(…)
L.append(object) -> None – append object to end

示例：

insert

在索引之前插入一个对象。

insert操作索引超出范围是：如果是正索引，等效append，如果是负索引，等效于insert(0,object)。

insert(…)
L.insert(index, object) – insert object before index

示例：

extend

通过添加一个可迭代对象（iterable）中的元素来扩展列表。

extend(…)
L.extend(iterable) -> None – extend list by appending elements from the iterable

示例：

删除元素

pop

删除并返回索引所在的元素（默认不填索引是指-1，最后一个）。

如果index超出索引范围，会抛出IndexError异常。

pop(…)
L.pop([index]) -> item – remove and return item at index (default last).
Raises IndexError if list is empty or index is out of range.

remove

删除第一次发现的值。

如果值不存在则抛出ValueError。

remove(…)
L.remove(value) -> None – remove first occurrence of value.
Raises ValueError if the value is not present.

clear

清空列表。

clear(…)
L.clear() -> None – remove all items from L

查找/统计元素

index

查找某值的第一个索引，可以指定索引的开始和结束位置。（包含start，不包含stop）

如果值没有找到，则抛出ValueError。

index(…)
L.index(value, [start, [stop]]) -> integer – return first index of value.
Raises ValueError if the value is not present.

count

计数，返回值出现的次数。

count(...)
L.count(value) -> integer – return number of occurrences of value

len函数

len函数可以用在列表、元组、字典上，输出容器中元素的个数。

len(obj, /)
Return the number of items in a container.

修改列表

sort

原地排序，有两个参数，一个是key，表示排序用的函数，默认为None没有。一个是reverse（反向，降序排列），默认是False。

sort(…)
L.sort(key=None, reverse=False) -> None – stable sort “IN PLACE”

reverse

原地反转。

reverse(…)
L.reverse() – reverse “IN PLACE”

其他

copy

采用赋值方法创建新列表，是指向原来的列表的那块内存，当进行更改时候，新老列表一起改变。

采用copy方法，就是复制出一块相同的内存(影子复制)。新旧列表互不影响。

copy(…)
L.copy() -> list – a shallow copy of L
