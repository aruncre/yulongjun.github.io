3.9 字典（Dictionarys）

发表于 2016-07-21   |   分类于 Learning-Python3   |  

字典是一种key-value结构。字典的key必须是可hash的值，并且是唯一的，value可以是任意值。

字典初始化

字典的下标操作

增

update

update的参数可以是以下几种情况：

  • 字典
  • 由二元组构成的可迭代对象
  • 关键字参数

update如果key相同的话，第二个的value会覆盖第一个的value，算是一个改的功能了。

setdefault

可以根据某个key来设置默认值，用到很少。

如果key已经存在的话，就更改不了。

defaultdict

设置默认值，默认值是一个函数。

灵活应用的部分（可选读，后期才会学函数。）

删

pop

根据key来删除键值对，并返回value。

如果不存在，则会抛出KeyError。

可以指定异常的返回值，这样抛出异常，就会返回那个返回值。

pop(…)
D.pop(k[,d]) -> v, remove specified key and return the corresponding value.
If key is not found, d is returned if given, otherwise KeyError is raised

popitem

随机删除并返回一个k-v对。（用得不多）

del函数

用的不多，pop已经能实现del函数的功能了。

clear

清空字典。

访问（查）

可以用下标访问，也可以用get，get存在的key，会返回value，get不存在的key，会返回空，可以设置不存在的key的默认的返回值。

get

遍历

keys遍历key

values遍历value

items遍历kv的二元组

其他几个函数

copy

复制字典，相当于复制了字典那块的内存。

fromkeys

fromkeys(iterable, value = None)
把可迭代对象中的值当做key，生成一组带默认值的字典。
